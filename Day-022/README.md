### ğŸ”¹ Class

### ğŸ”¹ Object

## ğŸ¯ Interview Preparation (Day 22 Level)

**Q1. What is a class?**  
A class is a user-defined data type that acts as a blueprint for creating objects.

**Q2. What is an object?**  
An object is an instance of a class that contains actual values and can access class methods.

**Q3. Where are objects stored in memory?**  
Objects are stored in heap memory.

**Q4. What does a reference variable store?**  
It stores the address of the object.

**Q5. Can we create multiple objects from one class?**  
Yes, and each object will have its own separate data.
