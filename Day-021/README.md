## ğŸ¯ Interview Preparation (Day 21 Level)

**Q1. What is Object-Oriented Programming?**  
OOP is a programming paradigm based on objects that contain data and methods.

**Q2. What are the four pillars of OOP?**  
Encapsulation, Inheritance, Polymorphism, and Abstraction.

**Q3. Why is OOP important?**  
It provides modularity, reusability, scalability, and better code organization.

**Q4. What is the difference between class and object?**  
A class is a blueprint, while an object is an instance of that class.
