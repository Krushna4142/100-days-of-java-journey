## Object-Oriented Programming

### Classes and Objects

Classes define structure and objects represent instances.

### Inheritance

Allows code reuse and hierarchical relationships.

### Polymorphism

Allows methods to behave differently depending on context.

### Encapsulation

Protects data using private variables and controlled access.

### Abstraction

Hides implementation details and exposes only functionality.

# 🔹 Exception Handling

Exception handling is used to manage runtime errors.

Important keywords:

- try
- catch
- finally
- throw
- throws

Benefits:

- Prevents program crashes
- Improves program reliability
- Handles unexpected errors

# 🔹 Collections Framework

Collections help manage groups of objects efficiently.

Common types:

### ArrayList

Dynamic list that allows duplicates.

### HashSet

Stores unique elements.

### HashMap

Stores key-value pairs.
