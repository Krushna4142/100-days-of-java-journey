## 🧩 Understanding Java Collections

The **Java Collections Framework** is a set of classes and interfaces that implement commonly reusable data structures.

It provides:

- Dynamic data storage
- Built-in algorithms
- Efficient searching and sorting

Collections are preferred over arrays because they:

- Can grow dynamically
- Provide built-in utility methods
- Simplify data manipulation

## 🎯 Interview Preparation (Day 28 Level)

**Q1. What is the Java Collections Framework?**  
It is a set of classes and interfaces that provide dynamic data structures and algorithms.

**Q2. What is the difference between Array and ArrayList?**  
Arrays have fixed size, while ArrayList can grow dynamically.

**Q3. What is the difference between List and Set?**  
List allows duplicates, Set does not.

**Q4. What is a Map in Java?**  
A Map stores data in key-value pairs.

**Q5. Which collection is commonly used in Java applications?**  
ArrayList and HashMap are widely used.
