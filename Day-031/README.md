## 🧩 Understanding ArrayList

`ArrayList` is a part of the **Java Collections Framework** and implements the **List interface**.

Unlike traditional arrays:

- ArrayList **automatically resizes**
- Allows **dynamic insertion and deletion**
- Provides **built-in utility methods**

Example difference:

| Feature          | Array               | ArrayList            |
| ---------------- | ------------------- | -------------------- |
| Size             | Fixed               | Dynamic              |
| Data Type        | Primitive & Objects | Objects only         |
| Built-in Methods | Limited             | Many utility methods |

## 🎯 Interview Preparation (Day 31 Level)

**Q1. What is ArrayList in Java?**  
ArrayList is a resizable array implementation of the List interface in the Java Collections Framework.

**Q2. What is the difference between Array and ArrayList?**  
Arrays have fixed size while ArrayList can grow or shrink dynamically.

**Q3. Can ArrayList store primitive data types?**  
No. It stores objects only. Primitive values are stored using wrapper classes like `Integer`, `Double`, etc.

**Q4. Which package contains ArrayList?**  
ArrayList is available in `java.util` package.
