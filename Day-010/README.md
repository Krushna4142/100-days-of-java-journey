## ğŸ¯ Interview Preparation (Day 10 Level)

**Q1. How do you take array input in Java?**  
Using `Scanner` and a loop.

**Q2. Can array size be dynamic in Java?**  
Size is fixed after creation, but it can be decided at runtime.

**Q3. Why use loops for array input?**  
Because array elements are accessed using index.

**Q4. What happens if index exceeds array length?**  
`ArrayIndexOutOfBoundsException`
