# 🧠 Day 030 – Mini Project Notes

## 🎯 Project: Student Record Management System

This project combines multiple Java concepts learned during the first 30 days.

The goal was to create a **simple console-based system** for managing student records.

---

# 🔹 Concepts Used

## Object-Oriented Programming

### Classes and Objects

The project defines a `Student` class representing student data.

### Encapsulation

Student fields are kept private and accessed using methods.

---

## Collections Framework

### ArrayList

Used to dynamically store student objects.

Benefits:

- Dynamic size
- Easy insertion and removal
- Built-in utility methods

---

## Control Flow

The program uses:

- `while` loop → to keep program running
- `switch` statement → menu navigation
- `if` conditions → logic handling

---

# 🔹 Program Flow

1. Program displays menu
2. User selects option
3. Based on input:
   - Add student
   - View students
   - Remove student
4. Program continues until exit

---

# 🔹 What This Project Demonstrates

This project demonstrates:

- Basic **system design thinking**
- How multiple Java concepts combine
- How to structure a small program
- How collections manage objects

---

Future improvements could include:
