## ğŸ¯ Interview Preparation (Day 016 Level)

**Q1. What is a 2D array?**  
A 2D array is an array of arrays used to store data in rows and columns.

**Q2. How do you traverse a 2D array?**  
Using nested loops where the outer loop controls rows and the inner loop controls columns.

**Q3. Where are 2D arrays used in real life?**  
In matrices, games, image processing, and table-based data.

**Q4. Difference between 1D and 2D array?**  
1D array stores data linearly, while 2D array stores data in tabular form.
