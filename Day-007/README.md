## ğŸ¯ Interview Preparation (Day 07 Level)

**Q1. What is a loop in Java?**  
A loop allows repeated execution of a block of code as long as a condition is true.

**Q2. Difference between for and while loop?**  
`for` loop is used when the number of iterations is known, while `while` loop is used when it depends on a condition.

**Q3. What is a do-while loop?**  
It is an exit-controlled loop that executes at least once before checking the condition.

**Q4. What is an infinite loop?**  
A loop that never terminates because its condition never becomes false.
