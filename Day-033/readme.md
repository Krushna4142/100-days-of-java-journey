## 🧩 Understanding CRUD Operations

CRUD operations form the foundation of most software systems.

| Operation | Meaning              |
| --------- | -------------------- |
| Create    | Add new data         |
| Read      | View existing data   |
| Update    | Modify existing data |
| Delete    | Remove data          |

Almost every application uses CRUD logic:

- Banking systems
- E-commerce apps
- Student management systems
- Social media platforms
